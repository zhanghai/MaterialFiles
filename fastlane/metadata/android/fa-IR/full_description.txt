یک مدیر پروندهٔ متن‌باز با طراحی متریال.

قابلیت‌ها:
- متن‌باز: سبُک، تمیز و ایمن.
- طراحی متریال: از رهنمودهای طراحی متریال، با توجه زیاد به جزییات پیروی می‌کند.
ـ Breadcrumbs: به راحتی در سامانه‌پرونده حرکت کنید.
- پشتیبانی از ریشه: نمایش و مدیریت پرونده‌ها با دسترسی ریشه.
- پشتیبانی از بایگانی: نمایش، استخراج و ایجاد پرونده‌های فشردهٔ رایج.
- پشتیبانی از NAS: نمایش و مدیریت پرونده‌های روی کارسازهای SFTP و SMB.
- زمینه‌ها: رنگ‌های رابط کاربری قابل سفارشی‌سازی، به اضافهٔ حالت شب با گزینهٔ سیاه واقعی.
- لینوکس آگاه: پیوندهای نمادین، مجوزهای پرونده و مفاد اس‌ای‌لینوکس را می‌داند.
- قدرتمند: از فراخوانی‌های سامانهٔ لینوکس در زیر کاپوت استفاده می‌کند، نه یک تجزیه‌کنندهٔ ls دیگر.
- به خوبی اجرا شده: بر اساس چیزهای درست، از جمله Java NIO2 File API و LiveData ساخته شده است.
